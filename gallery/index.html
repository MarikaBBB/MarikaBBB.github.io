My Project's Gallery

Project 1

This is a bookshelf web page featuring my top 3 favourite books. The page includes book cover images and information on the title, author, and year of publication for each book. The page also displays the current date and features a link to my Goodreads profile for anyone who wants to see what I am currently reading.

Project 2

This project is a web gallery showcasing a collection of my projects developed for the Founders & Coders application. The gallery is built using HTML, CSS, and JavaScript, and is designed to be responsive and adaptable to different screen sizes. The project serves as a showcase of the skills I am gaining in web development, as well as a demonstration of my ability to design responsive and user-friendly interfaces.
